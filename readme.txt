
Instructions

1. Copy all the python files.
2. Edit the varibale "path" in main file(swarm_robot_monitoring.py) to get the location of the video file.
3. run the main file(swarm_robot_monitoring.py).
4. Video will start to play.
5. Press 'c' to capture a test frame at a desired time.This will be used to set values for color filtering process.
6. Then the captured frame will display. Press any key to continue.
7. Change values in the interface and click "test for Robots/Arena" button to check whether selected values filter out the colour of the robots.
8. If the color filteration is acceptable click "set for robots" to store the values for color filtering of robots.
9. Do the same to set the values to filter the color of the arena.
*. recommended values for the sample video, provided for group B
	For robots,
		Channel 1 Lower limit-70
		Channel 1 upper limit - 170
		Channel 2 lower limit- 100
		Channel 2 upper limit - 190
		Channel 3 lower limit - 100
		Channel 3 upper limit - 230
	For Arena,
		Channel 1 Lower limit-0
		Channel 1 upper limit - 0
		Channel 2 lower limit- 0
		Channel 2 upper limit - 0
		Channel 3 lower limit - 239
		Channel 3 upper limit - 240
			
10. Click "proceed" button to continue.
